description: >
  <p>3Dスケッチにおいて、線の遠近法を有効または無効にします。</p>

  <p>WebGLモードでは、カメラから遠ざかるにつれて線をより細く描画することができます。これにより、より現実的な外観が得られます。</p>

  <p>デフォルトでは、使用されている遠近法の種類に基づいて線が異なる方法で描画されます：</p>

  <ul>

  <li><code>perspective()</code>と<code>frustum()</code>は現実的な遠近法をシミュレートします。これらのモードでは、線の太さは線とカメラの距離に影響されます。これにより、より自然な外観が得られます。<code>perspective()</code>は3Dスケッチのデフォルトモードです。</li>

  <li><code>ortho()</code>は現実的な遠近法をシミュレートしません。このモードでは、線の太さは線とカメラの距離に関係なく一定です。これにより、より予測可能で一貫した外観が得られます。</li>

  </ul>

  <p><code>linePerspective()</code>はデフォルトの線描画モードを上書きすることができます。</p>

  <p>パラメータ<code>enable</code>はオプションです。これは線の描画方法を設定する<code>Boolean</code>値です。<code>true</code>が渡された場合（<code>linePerspective(true)</code>のように）、線はカメラから遠ざかるにつれてより細く表示されます。<code>false</code>が渡された場合（<code>linePerspective(false)</code>のように）、線はカメラからの距離に関係なく一定の太さを持ちます。デフォルトでは、<code>linePerspective()</code>は有効になっています。</p>

  <p>引数を渡さずに<code>linePerspective()</code>を呼び出すと、有効な場合は<code>true</code>を、そうでない場合は<code>false</code>を返します。</p>

  <p>注意：<code>linePerspective()</code>はWebGLモードでのみ使用できます。</p>

    // キャンバスをダブルクリックして線の遠近法を切り替えます。

        'グレーの背景に黒い辺を持つ白い立方体。ユーザーがダブルクリックすると、その辺が太くなったり細くなったりします。'
      );

    function draw() {
      background(200);

      // 原点をカメラに向かって移動します。
      translate(-10, 10, 600);

      // 座標系を回転させます。
      rotateY(-0.1);
      rotateX(-0.1);

      // 立方体の列を描画します。
      for (let i = 0; i < 6; i += 1) {
        translate(0, 0, -40);
        box(10);
      }
    }

    // ユーザーがダブルクリックしたときに線の遠近法を切り替えます。
    function doubleClicked() {
      let isEnabled = linePerspective();
      linePerspective(!isEnabled);
    }
