description: >
  <p>楕円、円、弧の描画方法を変更します。</p>

  <p>デフォルトでは、<a href="/reference/p5/ellipse/">ellipse()</a>、<a
  href="/reference/p5/circle/">circle()</a>、

  <a href="/reference/p5/arc/">arc()</a>の最初の2つのパラメータは、

  図形の中心のx座標とy座標です。次のパラメータは

  図形の幅と高さを設定します。これは

  <code>ellipseMode(CENTER)</code>を呼び出すのと同じです。</p>

  <p><code>ellipseMode(RADIUS)</code>も最初の2つのパラメータを使用して

  図形の中心のx座標とy座標を設定します。次のパラメータは

  図形の幅と高さの半分です。<code>ellipse(0, 0, 10, 15)</code>を呼び出すと、

  幅20、高さ30の図形が描画されます。</p>

  <p><code>ellipseMode(CORNER)</code>は最初の2つのパラメータを

  図形の左上隅として使用します。次のパラメータはその幅と高さです。</p>

  <p><code>ellipseMode(CORNERS)</code>は最初の2つのパラメータを

  楕円の境界ボックスの一つの角の位置として使用します。次のパラメータは

  反対側の角の位置です。</p>

  <p><code>ellipseMode()</code>に渡される引数は、すべて大文字で書く必要があります。

  なぜなら、<code>CENTER</code>、<code>RADIUS</code>、<code>CORNER</code>、
  <code>CORNERS</code>の定数がこのように

  定義されているからです。JavaScriptは大文字と小文字を区別する言語です。</p>

  - |-

    <div>
    <code>
    function setup() {
      createCanvas(100, 100);

      background(200);

      // 白い楕円
      ellipseMode(RADIUS);
      fill(255);
      ellipse(50, 50, 30, 30);

      // グレーの楕円
      ellipseMode(CENTER);
      fill(100);
      ellipse(50, 50, 30, 30);

      describe('中心にグレーの円がある白い円。両方の円には黒い輪郭があります。');
    }
    </code>

      createCanvas(100, 100);

      background(200);

      // 白い楕円
      ellipseMode(CORNER);
      fill(255);
      ellipse(25, 25, 50, 50);

      // グレーの楕円
      ellipseMode(CORNERS);
      fill(100);
      ellipse(25, 25, 50, 50);

      describe('左上隅にグレーの円がある白い円。両方の円には黒い輪郭があります。');
    }
    </code>
    </div>
