description: >
  <p>コードのブロックを繰り返す方法。</p>

  <p><code>while</code>ループは、条件が<code>true</code>である間、文を繰り返すのに役立ちます。
  これは繰り返される<code>if</code>文のようなものです。例えば、<code>while</code>
  ループを使うと、「複数の線を描く」というアイデアを次のように簡単に表現できます：</p>

  <pre><code class="language-js">// 繰り返しを追跡する変数を宣言する
  let x = 10;


  // x < 100である限り繰り返す
  while (x < 100) {
    line(x, 25, x, 75);

    // 20ずつ増加させる
    x += 20;
  }

  </code></pre>

  <p>ループのヘッダーは<code>while</code>キーワードで始まります。ループは一般的に、
  繰り返しまたは反復するにつれて、カウントアップまたはカウントダウンします。
  括弧内の文<code>x < 100</code>は、ループが各反復で確認する条件です。
  条件が<code>true</code>の場合、ループは中括弧<code>{}</code>の間のコードを実行します。
  中括弧の間のコードはループの本体と呼ばれます。条件が<code>false</code>の場合、
  本体はスキップされ、ループは停止します。</p>

  <p>誤って無限ループを作成することはよくあります。例えば、
  次のループはカウントアップしないため、決して反復を停止しません：</p>

  <pre><code class="language-js">// 繰り返しを追跡する変数を宣言する
  let x = 10;


  // x < 100である限り繰り返す
  while (x < 100) {
    line(x, 25, x, 75);
  }


  // これはループの本体内にあるべきです！
  x += 20;

  </code></pre>

  <p>文<code>x += 20</code>がループの本体の後に表示されています。これは、
  変数<code>x</code>が<code>10</code>のままで、常に<code>100</code>未満であることを意味します。</p>

  <p><code>while</code>ループは、反復回数が事前にわからない場合に便利です。
  例えば、同心円をランダムな間隔で描くことができます：</p>

  <pre><code class="language-js">let d = 100;
  let minSize = 5;


  while (d > minSize) {
    circle(50, 50, d);
    d -= random(10);
  }

  </code></pre>
