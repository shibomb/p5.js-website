description: >
  <p>ベクトルの大きさ、または長さを計算します。</p>

  <p>ベクトルは異なる方法で考えることができます。一つの見方では、ベクトルは

  空間上の点です。ベクトルの成分、<code>x</code>と<code>y</code>は、点の

  座標<code>(x, y)</code>です。ベクトルの大きさは、原点

  <code>(0, 0)</code>から<code>(x, y)</code>までの距離です。<code>mag(x, y)</code>は

  <code>dist(0, 0, x, y)</code>を呼び出すための短縮形です。</p>

  <p>ベクトルは、空間内を指す矢印としても考えることができます。この見方は

  動きをプログラミングする際に役立ちます。詳細については<a
  href="/reference/p5/p5.Vector">p5.Vector</a>を

  参照してください。</p>

  <p><a href="/reference/p5/p5.Vector">p5.Vector</a>オブジェクトの

  大きさを計算するには、<a href="/reference/p5.Vector/mag/">p5.Vector.mag()</a>を

  使用してください。</p>

    function setup() {
