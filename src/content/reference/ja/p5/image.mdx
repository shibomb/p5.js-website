description: >
  <p>キャンバスに画像を描画します。</p>

  <p>最初のパラメータ<code>img</code>は、描画する元の画像です。
  <code>img</code>は以下のオブジェクトのいずれかになります：</p>

  <ul>

  <li><a href="/reference/p5/p5.Image">p5.Image</a></li>

  <li><a href="/reference/p5/p5.Element">p5.Element</a></li>

  <li><a href="/reference/p5/p5.Texture">p5.Texture</a></li>

  <li><a href="/reference/p5/p5.Framebuffer">p5.Framebuffer</a></li>

  <li><a
  href="/reference/p5/p5.FramebufferTexture">p5.FramebufferTexture</a></li>

  </ul>

  <p>2番目と3番目のパラメータ<code>dx</code>と<code>dy</code>は、
  描画先の画像の左上隅の座標を設定します。画像を配置する他の方法については、
  <a href="/reference/p5/imageMode/">imageMode()</a>を参照してください。</p>

  <p><code>image()</code>のオプションパラメータの動作を説明する図は以下の通りです：</p>

  <p><img src="/assets/drawImage.png"/></p>

  <p>4番目と5番目のパラメータ<code>dw</code>と<code>dh</code>はオプションです。これらは
  描画先の画像の幅と高さを設定します。デフォルトでは、<code>image()</code>は
  元の画像の全体を元のサイズで描画します。</p>

  <p>6番目と7番目のパラメータ<code>sx</code>と<code>sy</code>もオプションです。
  これらの座標は、元の画像から描画するサブセクションの左上隅を定義します。</p>

  <p>8番目と9番目のパラメータ<code>sw</code>と<code>sh</code>もオプションです。
  これらは元の画像から描画するサブセクションの幅と高さを定義します。
  デフォルトでは、<code>image()</code>は<code>(sx, sy)</code>から始まり、
  元の画像の端まで延びる全サブセクションを描画します。</p>

  <p>9番目のパラメータ<code>fit</code>もオプションです。これにより、元画像のサブセクションを
  アスペクト比に影響を与えずに描画することができます。<code>CONTAIN</code>が渡された場合、
  サブセクション全体が描画先の矩形内に表示されます。<code>COVER</code>が渡された場合、
  サブセクションは描画先の矩形を完全に覆います。これにより、サブセクションが
  ズームインされたような効果が得られる場合があります。</p>

  <p>10番目と11番目のパラメータ<code>xAlign</code>と<code>yAlign</code>もオプションです。
  これらは、フィットしたサブセクションの配置方法を決定します。<code>xAlign</code>は
  <code>LEFT</code>、<code>RIGHT</code>、または<code>CENTER</code>に設定できます。
  <code>yAlign</code>は<code>TOP</code>、<code>BOTTOM</code>、または<code>CENTER</code>に
  設定できます。デフォルトでは、<code>xAlign</code>と<code>yAlign</code>の両方が
  <code>CENTER</code>に設定されています。</p>
