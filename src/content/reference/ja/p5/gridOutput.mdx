description: >
  <p>キャンバス上の図形のスクリーンリーダーでアクセス可能な説明を作成します。</p>

  <p><code>gridOutput()</code>は、一般的な説明、図形の表、および図形のリストを

  ウェブページに追加します。一般的な説明には、キャンバスのサイズ、

  キャンバスの色、図形の数が含まれます。例えば、

  <code>グレーのキャンバス、100 x 100ピクセル、2つの図形を含む：円1つ、正方形1つ</code>。</p>

  <p><code>gridOutput()</code>は図形の表をグリッドとして使用します。グリッド内の各図形は、

  キャンバス上の図形の位置に対応する行と列のセルに配置されます。

  グリッドのセルは、その位置にある図形の色と種類を説明します。

  例えば、<code>赤い円</code>。これらの説明は個別に選択して

  より詳細な情報を得ることができます。これは

  <a href="/reference/p5/textOutput/">textOutput()</a>が表をリストとして使用するのとは異なります。</p>

  <p>図形のリストは表の後に続きます。リストは各図形の色、種類、

  位置、面積を説明します。例えば、

  <code>赤い円、位置 = 中央、面積 = 3%</code>。</p>

  <p><code>display</code>パラメータはオプションです。これは説明の

  表示方法を決定します。<code>LABEL</code>が渡された場合、例えば

  <code>gridOutput(LABEL)</code>のように、説明は

  キャンバスの隣のdiv要素に表示されます。<code>LABEL</code>を使用すると

  スクリーンリーダーにとって役に立たない重複が生じます。<code>LABEL</code>は

  開発中にのみ使用してください。<code>FALLBACK</code>が渡された場合、例えば

  <code>gridOutput(FALLBACK)</code>のように、説明は

  スクリーンリーダーにのみ表示されます。これがデフォルトの

  モードです。</p>

  <p>スケッチをアクセシブルにする方法の詳細については、

  <a href="/learn/accessible-labels.html">アクセシブルなキャンバスの説明を書く</a>

  をお読みください。</p>
